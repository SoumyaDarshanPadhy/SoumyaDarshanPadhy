### 🔭 What I Work On

- **Backend architecture** — Node.js/TypeScript services built on strict clean architecture (domain → application → repository → infrastructure), Express, and Prisma ORM over PostgreSQL
- **Enterprise auth & access control** — Azure AD/MSAL integration, Microsoft Graph API, and RBAC systems supporting base roles, custom roles, and fine-grained permission primitives
- **Data-intensive systems** — bulk sync pipelines, Excel import/export engines, and complex PostgreSQL migrations (partial indexes, CHECK constraints, enum evolution) on live, data-bearing databases
- **Cross-platform development** — Flutter (GetX) for mobile, React/TypeScript for admin dashboards, and native Android (Kotlin, MDM/Device Owner patterns) for embedded launcher apps
- **Infrastructure** — self-managed VPS environments with PostgreSQL, pgBouncer, pgBackRest, and Redis; designed scalable multi-tenant architectures (Socket.IO, BullMQ, Cloudflare R2)
